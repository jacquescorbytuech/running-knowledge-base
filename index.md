# Everything I Know — Running

The master catalogue. This is an evidence-graded knowledge base on distance running, biased toward what is real, science-based and proven, and honest about what is not. Start with [the basics](concepts/the-basics.md); it puts everything else in proportion.

Every claim-bearing page carries an `evidence:` grade (`strong` / `moderate` / `limited` / `weak` / `contested`) and cites its sources at the point of each claim. See [CLAUDE.md](CLAUDE.md) for the schema and editorial stance.

## Start here

- [The basics (what actually matters)](concepts/the-basics.md) — run a lot, mostly easy; sleep well; eat a balanced diet; be consistent. Almost everything else is the last few per cent.
- [Running for beginners](concepts/running-for-beginners.md) — how to start from scratch and stay uninjured.
- [Running for intermediates](concepts/running-for-intermediates.md) — the next stage, once you can run comfortably for half an hour.
- [Individual variation](concepts/individual-variation.md) — why even good average evidence is only a starting point for you.

Most [running injuries](injury/running-injuries.md) are overuse from training-load errors: [bone stress injuries](injury/bone-stress-injuries.md), [tendinopathy](injury/tendinopathy.md) (including [Achilles](injury/achilles-tendinopathy.md)), [patellofemoral pain](injury/patellofemoral-pain.md), [plantar fasciopathy](injury/plantar-fasciopathy.md), [calf and hamstring strains](injury/muscle-strains.md), the remaining [common overuse injuries](injury/common-overuse-injuries.md) (iliotibial band, shin splints), and the everyday [practical niggles](injury/practical-niggles.md). General knowledge, not medical advice.
