# Galloway, the run-walk-run method

Jeff Galloway's description of the run-walk-run method: planned walk breaks taken from the start of a run or race, before fatigue, with the run-to-walk ratio set per runner. The method's strongest injury-reduction and time-improvement claims come from Galloway's own coaching data rather than controlled trials, so treat them as anecdotal. Limited (practitioner source).
