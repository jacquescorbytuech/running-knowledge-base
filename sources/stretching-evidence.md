# Stretching, performance and running economy (scoping review)

Scoping review on stretching in runners: the paper's own finding is that long static stretches (≥60 s) impair strength and power by −4.6%, while prior reviews it cites report a wider range of −4.0 to −7.5%; short static stretches (<60 s) cause only trivial (~1.1%) decrements; effects on running economy are small and inconsistent. Dynamic warm-up is generally preferred before running. Static flexibility is largely unrelated, or slightly negatively related, to running economy. Moderate.
